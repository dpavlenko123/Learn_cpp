### Задача: реализовать лексер и парсер.

Результат: абстрактное синтаксическое дерево.

Поддерживаемые операции:
    +, -, *, /
    >, <, ==, <=, >=, !=

```
in a; //чтение из стандартного потока

b = 3 + a*5;
c = 20;
while(a < 10) {
    if(b < 6) {
        c = 15;
    }
    a = a + 1;
}

print a; //вывод в стандартный потока
```
