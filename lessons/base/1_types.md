## Базовые типы

---

**Base**

### 1. Целые типы (Integer types)

- `char` — символьный тип (см. раздел про `char` ниже).
- `short`, `int`, `long`, `long long` — целые знаковые типы разных размеров.
- Беззнаковые версии: `unsigned short`, `unsigned int`, `unsigned long`, `unsigned long long`.

Пример объявлений:

```c++
int i = -42;
unsigned int u = 42u;
long long big = 123456789012345LL;
```

### 2. `char` и `bool`

- `char` — хранит один байт, обычно используется для символов и байт-строк. Есть три разновидности: `char`, `signed char`, `unsigned char`.
- `bool` — булев тип; принимает значения `true` или `false`.

Примеры:

```c++
char c = 'A';         // символ (код символа сохраняется в одном байте)
bool flag = true;
```

### 3. Типы с плавающей точкой

- `float` — одинарная точность (обычно 32 бита).
- `double` — двойная точность (обычно 64 бита) — стандартный выбор для дробных чисел.
- `long double` — расширенная точность (платформенно-зависимая).

Пример литералов:

```c++
float f = 3.14f;
double d = 2.718281828;
long double ld = 1.0L;
```

### 4. Типы фиксированной ширины

- `int8_t`, `int16_t`, `int32_t`, `int64_t` и их беззнаковые аналоги `uint8_t`, ... — используются когда нужен гарантированный размер.
- `size_t` — беззнаковый тип для размеров и индексов (возвращается `sizeof`).

---

**Advance**

### 5. Размеры, диапазоны и `std::numeric_limits`

Для точной информации о размерах и диапазонах используйте заголовок `<limits>` и `std::numeric_limits<T>`; `sizeof(T)` даёт размер в байтах.

Пример получения информации о типе:

```c++
#include <iostream>
#include <limits>

std::cout << "int: " << sizeof(int) << " bytes, range: "
        << std::numeric_limits<int>::min() << ".." << std::numeric_limits<int>::max() << '\n';
```

### 6. `char` — детали и кодировки

- `char` представляет один байт памяти и хранит код символа (обычно UTF-8 байт или ASCII).
- Различие `char` vs `signed char` vs `unsigned char` влияет на интерпретацию битов (знаковость не фиксирована стандартом для plain `char`).
- Для юникод-символов используются `char16_t`, `char32_t` и `wchar_t` (платформенно-зависимые длины).

### 7. `bool` — реализация и использование

- `bool` хранит логические значения `true`/`false`. Размер может быть 1 байт (или больше), реализация зависит от компилятора.
- Используйте `bool` для флагов и условий; для компактного хранения битовых флагов рассмотрите `std::bitset`.

### 8. Вещественные типы — диапазоны и специальные значения

- `std::numeric_limits<T>` позволяет узнать `lowest()`, `max()`, `infinity()` и `quiet_NaN()` для `float`/`double`/`long double`.

Пример получения специальных значений:

```c++
#include <limits>
double inf = std::numeric_limits<double>::infinity();
double nan = std::numeric_limits<double>::quiet_NaN();
```

### 9. Когда какой тип выбирать

- Для счётчиков/индексов: `size_t`.
- Для целочисленной логики без особых требований: `int`.
- Для переносимости по размерам: `int32_t`/`int64_t`.
- Для дробных вычислений: `double` по умолчанию.
